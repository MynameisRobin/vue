# vue

> 基于vue2.0↑ + Element-UI 构建的后台管理系统

## 注意事项

``` bash
# 安装所有包
cnpm i
# 本地开发
npm run dev
# 本地构建
npm run build
# 构建到生产环境并查看打包分析报告
npm run build --report
```
 [脚手架指南](http://vuejs-templates.github.io/webpack/) 和 [vue-loader文档](http://vuejs.github.io/vue-loader).
